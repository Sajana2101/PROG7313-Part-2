# BudgetQuest

BudgetQuest is a personal budgeting Android application developed in Kotlin using Android Studio and Room Database. The app helps users manage their finances by allowing them to create spending categories, set monthly goals, track expenses, and upload receipt images.

---

## Features

- User registration and login
- Create custom expense categories
- Set category spending limits
- Set monthly budgeting goals
- Add expenses with:
  - category
  - amount
  - date
  - start time
  - end time
  - description
  - receipt image upload
- View expenses by category
- Edit expenses
- Delete expenses
- View uploaded receipt images in full screen
- User-specific data storage (each user only sees their own data)
- Logout functionality

---

## Technologies Used

- Kotlin
- Android Studio
- Room Database
- XML
- GitHub
- JUnit Testing

---

## How to Run the Project

1. Clone the repository
2. Open the project in Android Studio
3. Sync Gradle files
4. Run the application using an emulator or Android device
